# TRIGNUM-300M Notebooks

This directory contains Jupyter notebooks for interactive demonstrations.

## Planned Notebooks

- `01_introduction.ipynb` — Introduction to the Trignum Framework
- `02_magnetic_trillage.ipynb` — Interactive Magnetic Trillage visualization
- `03_tchip_demo.ipynb` — T-CHIP glow state demonstrations
- `04_subtractive_filter.ipynb` — Hallucination detection experiments

## Running

```bash
pip install jupyter
jupyter notebook
```
